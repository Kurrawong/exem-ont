depiction op

IRI http://xmlns.com/foaf/0.1/depiction
Is Defined By foaf:
Description

A depiction of some thing
Scope Note

Use this property to indicate an image that is the content of the example. The image must be available at a URL
Example 
# an example of using foaf:depiction to link to an image
# for the imaginary property ex:hasNumberOfLegs
PREFIX : <https://linked.data.gov.au/def/exem/>
PREFIX dcterms: <http://purl.org/dc/terms/>
PREFIX ex: <http://example.com/>
PREFIX foaf: <http://xmlns.com/foaf/0.1/>
PREFIX prof: <http://www.w3.org/ns/dx/prof/>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX sdo: <https://schema.org/>

ex:hasNumberOfLegs
    a rdf:Property ;
    :example [
        sdo:name "Remote image of legs" ;
        foaf:depiction "http://some-image-online.com/img/milli" ;
        dcterms:format "image/png" ;
    ] ;
.

Description
Example 
# an RDF Turtle example of the use of this property
# for the imaginary property ex:hasNumberOfLegs
PREFIX : <https://linked.data.gov.au/def/exem/>
PREFIX dcterms: <http://purl.org/dc/terms/>
PREFIX egroles: <https://linked.data.gov.au/def/exem/roles/>
PREFIX ex: <http://example.com/>
PREFIX prof: <http://www.w3.org/ns/dx/prof/>
PREFIX rdf: <http://www.w3.org/1999/02/22-rdf-syntax-ns#>
PREFIX sdo: <https://schema.org/>

ex:hasNumberOfLegs
    a rdf:Property ;
    :example [
        a prof:ResourceDescriptor ;
        sdo:name "Simple RDF Example" ;
        rdf:value '''ex:millipede-a
    a ex:Animal ;
    :hasNumberOfLegs 120 ;
.''' ;
        dcterms:format "text/turtle" ;
        prof:hasRole egroles:CanonicalExample ;
    ] ;
.

Source ap

IRI http://purl.org/dc/terms/source
Is Defined By DCMI Metadata Terms
Description

A related resource from which the described resource is derived.
Scope Note

Use this property to indicate a non-RDF resource, such as a web page, from wich this example was derived. Use prov:wasDerivedFrom to indicate an RDF resource. Use skos:historyNote to describe a non-linkable source or to contain a general description of how this example was created.

Use a datatype of xsd:anyURI for web pages
